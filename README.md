This project uses ESP32 and MPU6050 accelerometer to detect road conditions such as smooth surfaces, rough terrain, and potholes in real time.

Hardware

ESP32
MPU6050 Accelerometer
Breadboard & Wires

Features

Real-time vibration sensing
Road condition classification
Embedded C/C++ implementation

Working

The MPU6050 measures acceleration in three axes. Based on vibration intensity, the system classifies the road as smooth, rough, or pothole.
